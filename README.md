## 📖 Descripción del Proyecto
Este portal ha sido diseñado como la vitrina comercial y máquina de ventas automatizada para la agencia de desarrollo de software **Nexora Digital**. Su objetivo principal es permitir a los clientes contactar a la agencia, adquirir licencias de software propietario (como AutoZajuna Pro), y solicitar el desarrollo a medida de nuevas aplicaciones.

## ⚙️ Stack Técnico y Arquitectura
El portal está construido utilizando tecnologías web puras para garantizar máxima velocidad, SEO y portabilidad, alojado sin costos mediante arquitecturas Serverless/Estáticas.

- **Frontend:** HTML5 Semántico, Vanilla CSS (Variables, Flexbox, CSS Grid), Vanilla JavaScript (Animaciones, Validaciones).
- **Iconografía y Fuentes:** FontAwesome (Librería de iconos) y Google Fonts (Outfit).
- **Sistema de Correos:** EmailJS para el procesamiento de leads directamente desde el frontend sin necesidad de backend intermedio.
- **Hosting:** GitHub Pages (Despliegue automatizado vinculado a la rama `main`).

## 🛡️ Medidas de Seguridad Implementadas (DevSecOps)
El portal ha sido blindado contra ataques comunes y extracción de datos masivos (Scraping/Infostealers):
1. **Política de Seguridad de Contenido (CSP):** Prevención de inyección de código malicioso (XSS).
2. **Anti-Clickjacking:** Configuración estricta de `X-Frame-Options: DENY` y `frame-ancestors 'none'`.
3. **Ofuscación de Formularios (Anti-Infostealer):** Identificadores como `name` e `id` en el formulario han sido ofuscados (ej. `nxd_e_val` en lugar de `email`) para evitar bots de recolección automática de datos.

---

## 📈 Conversion Rate Optimization (CRO) & SEO
1. **Exit-Intent Lead Magnet:** Modal programado en JavaScript que detecta cuando el usuario mueve el cursor fuera del viewport y le ofrece una "Auditoría Gratuita de 30 Minutos", logrando captar leads al borde del abandono.
2. **Banda de Confianza:** Inclusión visual de tecnologías de grado empresarial y escudos de garantía para elevar la autoridad.
3. **Sticky CTA (Móviles):** Botón permanente flotante inferior para "Solicitar Presupuesto" que reduce la fricción en navegadores móviles.
4. **GEO & AEO (Generative / Answer Engine Optimization):** Estructuras semánticas JSON-LD para que los bots de IA entiendan perfectamente la oferta comercial y los servicios.

## ⚖️ Aspectos Legales
Se incluye una página de `legales.html` rigurosa (basada en el entorno legal de Colombia), la cual comprende:
- **Términos y Condiciones:** Regulación estricta sobre el límite de indemnización por servicios contratados.
- **Política de Privacidad (Ley 1581 de 2012):** Protección y tratamiento de datos personales para la recolección de Leads.
- **Acuerdo de Exención de Responsabilidad para RPA:** Cláusula para proteger a Nexora Digital contra posibles bloqueos o daños resultantes del uso de APIs y automatizaciones en plataformas de terceros (Meta, Zajuna, etc.).
