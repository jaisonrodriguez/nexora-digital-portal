E-Commerce con Supabase: caso real de ArtDesign Tienda

Las plataformas como Shopify o WordPress se quedan cortas cuando el modelo de negocio requiere funcionalidades altamente interactivas antes de que el usuario agregue el producto al carrito. Así fue el reto con ArtDesign Tienda.

El Reto del Editor en Tiempo Real

El cliente vende prendas de vestir que el usuario final debe poder personalizar subiendo sus propios logos, añadiendo texto y modificando colores. Para esto, integramos Fabric.js, una potente librería de manipulación de canvas HTML5 que permitió a los usuarios diseñar sobre los mockups 3D de las camisetas.

La arquitectura: React 19 + Supabase

Decidimos usar el ecosistema de Supabase como Backend-as-a-Service (BaaS) por tres motivos fundamentales:

  • PostgreSQL Nativo: Nos permitió estructurar catálogos complejos con relaciones JSONB para variaciones de tallas y colores.
  • Storage: Los diseños generados en el canvas se exportan en alta resolución y se guardan directamente en los buckets de Supabase Storage mediante políticas seguras.
  • Row Level Security (RLS): Aseguramos que un usuario solo pueda ver sus propios diseños y pedidos, mientras que el rol `ADMIN` tiene acceso a todo el tablero de control.

Pagos con Wompi

Para procesar los pagos en Colombia, implementamos el Widget de Wompi (Bancolombia) que maneja Nequi, PSE y Tarjetas de Crédito de forma segura y sin salir del portal.
